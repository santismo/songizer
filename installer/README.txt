Songizer Suite for Logic Pro

1. Double-click “Install Songizer Suite.command”.
2. Restart Logic Pro when the installer finishes.
3. If Repeatizer is not listed right away, open Repeatizer once from ~/Applications, then restart Logic Pro.

The installer places the Audio Unit components in:
~/Library/Audio/Plug-Ins/Components

It places the Repeatizer AUv3 container app in:
~/Applications

Existing copies are moved to a timestamped backup in:
~/Library/Application Support/Songizer Suite/Backups

Repeatizer.app is included because macOS uses it to register the Repeatizer MIDI FX extension with Logic Pro.

The included components are ad-hoc signed. If macOS asks for confirmation, open the installer from Finder and follow its prompt.
